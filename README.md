
Flowpen
===

An experiment in a user interface for exploring fluid flow.

See it at: [flowpen.com](www.flowpen.com)

Develop
---

Install global tools:

	sudo npm install -g browserify gulp babel

Install dependencies:

	npm install

Link app in node_modules

	cd node_modules
	ln -s ../app .

Running:

	npm start
